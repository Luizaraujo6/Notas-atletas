# Notas-atletas
Este projeto é uma aplicação em JavaScript desenvolvida para calcular a média final de atletas.

# Sobre o Projeto

O objetivo era criar um sistema que recebesse o nome de vários atletas e suas cinco notas (dadas por cinco jurados diferentes). A regra da competição determina que a maior e a menor nota devem ser descartadas, e a média é calculada apenas com as três notas centrais.

# Como Funciona

O código funciona como uma "máquina de filtragem" para garantir uma nota justa. Ele pega a lista bagunçada de notas de cada atleta e organiza tudo antes de fazer a conta.
